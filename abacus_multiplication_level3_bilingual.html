📍 Column Names on the Abacus

📍 ಅಬ್ಯಾಕಸ್‌ನ ಕಾಲಮ್ ಹೆಸರುಗಳು

Each column on the abacus has a special name and place value. In multiplication, we always start from the middle pointer and move LEFT!

🌟 The 3 Golden Rules of Abacus Multiplication
1

🔢 Count the digits FIRST!

🔢 ಮೊದಲು ಅಂಕಿ ಎಣಿಸಿ!

Before you start, count how many digits are in the number you are multiplying (NOT the multiplier). This tells you which column to start from!

Example: 319 × 8 → 319 has 4 digits (3, 1, 9, and 8 as multiplier → count = 4)

ಶುರು ಮಾಡುವ ಮೊದಲು, ನೀವು ಗುಣಿಸುತ್ತಿರುವ ಸಂಖ್ಯೆಯಲ್ಲಿ ಎಷ್ಟು ಅಂಕಿ ಇವೆ ಎಣಿಸಿ.

ಉದಾ: 319 × 8 → 319 ರಲ್ಲಿ ೪ ಅಂಕಿ (3, 1, 9 ಮತ್ತು 8 = ಒಟ್ಟು ೪)

2

📍 Find your starting column!

📍 ಶುರುವಾಗುವ ಕಾಲಮ್ ಹುಡುಕಿ!

From the middle pointer, count LEFT that many places. That's where you start writing answers!

If digits = 4 → go 4 places LEFT from middle → start at 4th column

ಮಧ್ಯದ ಸೂಚಿಯಿಂದ ಎಡಕ್ಕೆ ಅಷ್ಟು ಕಾಲಮ್ ಹೋಗಿ. ಅಲ್ಲಿಂದ ಉತ್ತರ ಬರೆಯಲು ಶುರು!

ಅಂಕಿ = ೪ → ಮಧ್ಯದಿಂದ ೪ ಎಡಕ್ಕೆ → ೪ನೇ ಕಾಲಮ್ನಿಂದ ಶುರು

3

✍️ Write each product with 2 digits!

✍️ ಪ್ರತಿ ಉತ್ತರ ೨ ಅಂಕಿಯಲ್ಲಿ ಬರೆಯಿರಿ!

Multiply each digit and write the answer using 2 digits always!
If answer is 8 → write 0 8 (not just 8)
If answer is 24 → write 2 4 (tens place then units place)

ಪ್ರತಿ ಅಂಕಿ ಗುಣಿಸಿ ಉತ್ತರವನ್ನು ಯಾವಾಗಲೂ ೨ ಅಂಕಿಯಲ್ಲಿ ಬರೆಯಿರಿ!
ಉತ್ತರ 8 → 0 8 ಎಂದು ಬರೆಯಿರಿ
ಉತ್ತರ 24 → 2 4 ಎಂದು ಬರೆಯಿರಿ

🔥 Example 1: 319 × 8 = 2552
🍎 Let's solve 319 × 8 step by step!319 × 8 ಹಂತ ಹಂತವಾಗಿ ಬಿಡಿಸೋಣ!

📋 Setup: Count digits 📋 ಸಿದ್ಧತೆ: ಅಂಕಿ ಎಣಿಕೆ

319 × 8 → digits in multiplication = 4 (3, 1, 9, and multiplier 8)
Go 4 places LEFT from middle pointer → Start at 4th column (Thousands)
Each digit product goes into 2 columns at a time.

319 × 8 → ಗುಣಾಕಾರದಲ್ಲಿ ಅಂಕಿ = ೪ (3, 1, 9 ಮತ್ತು ಗುಣಕ 8)
ಮಧ್ಯದಿಂದ ೪ ಎಡಕ್ಕೆ → ೪ನೇ ಕಾಲಮ್ (ಸಾವಿರ)ದಿಂದ ಶುರು
ಪ್ರತಿ ಅಂಕಿ ಉತ್ತರ ೨ ಕಾಲಮ್‌ಗಳಲ್ಲಿ ಹೋಗ್ತದೆ.

1
Multiply first digit: 3 × 8 = 24 ಮೊದಲ ಅಂಕಿ ಗುಣಿಸಿ: 3 × 8 = 24
Write 2 on 4th column, 4 on 3rd column 4ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ 2, 3ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ 4 ಬರೆಯಿರಿ
24 → 24
2
Multiply second digit: 1 × 8 = 8 ಎರಡನೇ ಅಂಕಿ ಗುಣಿಸಿ: 1 × 8 = 8
Write 0 on 3rd column, 8 on 2nd column
(0 means nothing to add on 3rd column — skip it!) 3ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ 0, 2ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ 8 ಬರೆಯಿರಿ
(0 ಅಂದರೆ ಏನೂ ಸೇರಿಸಬೇಕಿಲ್ಲ!)
08 → 08
3
Multiply third digit: 9 × 8 = 72 ಮೂರನೇ ಅಂಕಿ ಗುಣಿಸಿ: 9 × 8 = 72
Write 7 on 2nd column, 2 on 1st column
But wait! 2nd column already has 8. So: 8 + 7 = 15!
Can't add 7 directly → use Big Brother: +3 = −3 + 10
So: add 3 to 2nd column (8+3=11→carry), then add 10 to 3rd column 2ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ 7, 1ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ 2 ಬರೆಯಿರಿ
ಆದರೆ 2ನೇ ಕಾಲಮ್‌ನಲ್ಲಿ ಈಗಾಗಲೇ 8 ಇದೆ! 8+7=15!
7 ನೇರ ಸೇರಿಸಲಾಗದು → ದೊಡ್ಡಣ್ಣ: +3 = −3 + 10 ಬಳಸಿ
72 → 72
✅ Final Answer on Abacus = 2552
✅ ಅಬ್ಯಾಕಸ್‌ನಲ್ಲಿ ಅಂತಿಮ ಉತ್ತರ = 2552
319 × 8 = 🎉 2552

⚡ Super Quick Summary!
1️⃣

Count Digits First

ಮೊದಲು ಅಂಕಿ ಎಣಿಸಿ

Count ALL digits including multiplier. That number = how far LEFT to go!

ಗುಣಕ ಸೇರಿ ಎಲ್ಲ ಅಂಕಿ ಎಣಿಸಿ. ಆ ಸಂಖ್ಯೆ = ಎಷ್ಟು ಎಡಕ್ಕೆ ಹೋಗಬೇಕು!

✌️

Always Write 2 Digits

ಯಾವಾಗಲೂ ೨ ಅಂಕಿ ಬರೆಯಿರಿ

8 → write 0 8. 24 → write 2 4. Never skip the zero!

8 → 0 8 ಬರೆಯಿರಿ. 24 → 2 4 ಬರೆಯಿರಿ. ಶೂನ್ಯ ಬಿಡಬೇಡಿ!

👫

Use Big Brother!

ದೊಡ್ಡಣ್ಣ ಬಳಸಿ!

If adding gets too full — use Big Brother formula to carry over!

ಸೇರಿಸಲು ಜಾಗ ಇಲ್ಲದಿದ್ದರೆ — ದೊಡ್ಡಣ್ಣ ಸೂತ್ರ ಬಳಸಿ!

⬅️

Always Go Left

ಯಾವಾಗಲೂ ಎಡಕ್ಕೆ

Answers are written LEFT to RIGHT, one pair of digits per column pair!

ಉತ್ತರ ಎಡದಿಂದ ಬಲಕ್ಕೆ, ಒಂದು ಜೋಡಿ ಅಂಕಿ ಒಂದು ಜೋಡಿ ಕಾಲಮ್!

🔢

Each Digit Gets 2 Columns

ಪ್ರತಿ ಅಂಕಿಗೆ ೨ ಕಾಲಮ್

Tens digit of product → left column. Units digit → right column.

ಉತ್ತರದ ಹತ್ತರ ಅಂಕಿ → ಎಡ ಕಾಲಮ್. ಒಂದರ ಅಂಕಿ → ಬಲ ಕಾಲಮ್.

🎯

Add to Existing Beads

ಇರುವ ಮಣಿಗೆ ಸೇರಿಸಿ

New digits are ADDED to whatever beads are already on that column!

ಹೊಸ ಅಂಕಿಗಳನ್ನು ಆ ಕಾಲಮ್‌ನಲ್ಲಿ ಈಗಾಗಲೇ ಇರುವ ಮಣಿಗೆ ಸೇರಿಸಿ!

💡 Tips to Remember!
🎵

Sing the Tables!

ಟೇಬಲ್ ಹಾಡಿ!

Knowing 1–9 times tables perfectly makes abacus multiplication super fast!

1–9 ಟೇಬಲ್ ಕಂಠಪಾಠ ಮಾಡಿದರೆ ಗುಣಾಕಾರ ಬಹಳ ವೇಗ!

✌️

Two Columns Per Digit

ಒಂದು ಅಂಕಿ = ೨ ಕಾಲಮ್

Each multiplication product takes 2 columns. Never forget the zero!

ಪ್ರತಿ ಗುಣಾಕಾರ ಉತ್ತರ ೨ ಕಾಲಮ್ ತೆಗೆದುಕೊಳ್ಳ್ತದೆ. ಶೂನ್ಯ ಮರೆಯಬೇಡಿ!

📍

Count Left from Middle

ಮಧ್ಯದಿಂದ ಎಡಕ್ಕೆ

Always find your starting column by counting left from the middle pointer!

ಮಧ್ಯದ ಸೂಚಿಯಿಂದ ಎಡಕ್ಕೆ ಎಣಿಸಿ ಯಾವ ಕಾಲಮ್‌ನಲ್ಲಿ ಶುರು ಮಾಡಬೇಕು ಹುಡುಕಿ!

➕

Add, Don't Replace!

ಸೇರಿಸಿ, ಬದಲಿಸಬೇಡಿ!

New digits are always ADDED to existing beads. Never remove old beads!

ಹೊಸ ಅಂಕಿಗಳನ್ನು ಇರುವ ಮಣಿಗೆ ಸೇರಿಸಿ. ಹಳೆ ಮಣಿ ತೆಗೆಯಬೇಡಿ!

👫

Big Brother Helps!

ದೊಡ್ಡಣ್ಣ ಸಹಾಯ!

When column is full during adding, use Big Brother to carry to next column!

ಕಾಲಮ್ ತುಂಬಿದರೆ ದೊಡ್ಡಣ್ಣ ಸೂತ್ರ ಬಳಸಿ ಮುಂದಿನ ಕಾಲಮ್‌ಗೆ ಸಾಗಿಸಿ!

🔄

ಪ್ರತಿದಿನ ಅಭ್ಯಾಸ!

ಪ್ರತಿದಿನ ೫ ಗುಣಾಕಾರ ಅಭ್ಯಾಸ ಮಾಡಿ — ಮಿಂಚಿನ ವೇಗ ಸಿಗ್ತದೆ! ⚡
