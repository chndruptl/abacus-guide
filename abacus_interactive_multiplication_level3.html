📋 Multiplication Rules — Quick Reference

📋 ಗುಣಾಕಾರ ನಿಯಮಗಳು — ತ್ವರಿತ ಉಲ್ಲೇಖ

① Count Digits① ಅಂಕಿ ಎಣಿಸಿ
Count all digits incl. multiplierಗುಣಕ ಸೇರಿ ಎಲ್ಲ ಅಂಕಿ ಎಣಿಸಿ
② Go Left② ಎಡಕ್ಕೆ ಹೋಗಿ
Count left from middle pointerಮಧ್ಯದಿಂದ ಎಡಕ್ಕೆ ಎಣಿಸಿ
③ Write 2 Digits③ ೨ ಅಂಕಿ ಬರೆಯಿರಿ
8 → write 0 8, never skip zero!8 → 0 8 ಬರೆಯಿರಿ, ಶೂನ್ಯ ಬಿಡಬೇಡಿ!
④ Add, Don't Replace④ ಸೇರಿಸಿ, ಬದಲಿಸಬೇಡಿ
New digits added to existing beadsಹೊಸ ಅಂಕಿ ಇರುವ ಮಣಿಗೆ ಸೇರಿಸಿ
⑤ Big Brother Carry⑤ ದೊಡ್ಡಣ್ಣ ಕ್ಯಾರಿ
Column full? Use Big Brother formula!ಕಾಲಮ್ ತುಂಬಿದರೆ ದೊಡ್ಡಣ್ಣ ಸೂತ್ರ!
⑥ Shift Right Each Digit⑥ ಪ್ರತಿ ಅಂಕಿ ಒಂದು ಬಲಕ್ಕೆ
Each new multiplicand digit → 1 col rightಪ್ರತಿ ಹೊಸ ಅಂಕಿ → ೧ ಕಾಲಮ್ ಬಲಕ್ಕೆ
